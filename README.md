# IceTech Dashboard

Prototipo para registro de patinadores, asociación de códigos QR e inspección técnica de patines de hockey y patinaje artístico.

## Descripción

IceTech Dashboard es una aplicación web desarrollada con React, Vite y TypeScript.
El sistema permite registrar patinadores, capturar datos personales, asociar códigos QR, registrar información técnica del patín y llenar formularios de inspección para hockey y patinaje artístico.

## Requisitos

Antes de correr el proyecto necesitas tener instalado:

* Node.js
* npm

## Instalación

Clonar el repositorio:

```bash
git clone https://github.com/andreauxue/ice-tech-dashboard.git
```

Entrar a la carpeta del proyecto:

```bash
cd ice-tech-dashboard
```

Instalar dependencias:

```bash
npm install
```

## Ejecutar en desarrollo

Para iniciar el proyecto en modo desarrollo:

```bash
npm run dev
```

Después abre en el navegador la URL que aparezca en la terminal, normalmente:

```bash
http://localhost:5173
```

## Comandos principales

Instalar dependencias:

```bash
npm install
```

Ejecutar el servidor de desarrollo:

```bash
npm run dev
```

Generar versión de producción:

```bash
npm run build
```

Previsualizar la versión de producción:

```bash
npm run preview
```
